5. 部署配置

Cloudflare Pages 建议配置

Framework preset: Astro

Build command: npm run build

Build output directory: dist

Environment variables: 无需特殊变量

自定义域名: 在 Pages 项目设置中添加 a-zan.xyz 和 blog.a-zan.xyz，按提示配置 DNS CNAME

路由: Pages 自动按 Host 服务同一份静态文件，无需 Workers 重写
